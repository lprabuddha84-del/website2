# Lanka Luxe | Premium Sri Lankan Fashion E-Commerce Website

This folder contains the source code for the **Lanka Luxe** premium online shop website.

## 🎨 Design Theme: "Tropical Luxury"
*   **Colors**: Forest Emerald (`#0F3A2E`), Champagne Gold (`#C5A880`), Warm Alabaster (`#FAF8F5`), Charcoal Black (`#1C1E1D`).
*   **Typography**: Playfair Display (headings) & Plus Jakarta Sans (body).

## 📁 File Structure
*   `index.html` - Homepage with collection highlights, brand history, and trending carousel.
*   `shop.html` - Advanced catalog with real-time search, sidebar filters, and sorting.
*   `product.html` - Single product detail page with image switching, hover zoom, size/color variant swatches, and review form submissions.
*   `wishlist.html` - Dedicated wishlist space to review and manage favorited items.
*   `checkout.html` - Secure 3-step checkout with real-time price calculations and receipt generation.
*   `script.js` - E-commerce engine handling cart/wishlist localStorage sharing and the slide-out cart drawer.
